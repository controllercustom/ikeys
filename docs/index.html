Instructions

  1. Connect your ESP32-S3 board to your computer via USB
  2. Put the board into download mode (AtomS3: hold Reset button 2–3 seconds until green LED)
  3. Select your board below, then click Connect and choose it from the browser dialog
  4. Click Install to flash the firmware
  5. Wait for the flash to complete — the board reboots automatically
Note: WebSerial requires Chrome or Edge on desktop or Android. Safari and Firefox are not supported.

About iKeys

Wireless assistive keyboard that turns a tablet or phone into a full USB keyboard and mouse for any computer. Uses an ESP32-S3 with native USB HID — no software to install on the host.

  • 8×12 CSS grid keyboard + numpad overlay
  • Mouse emulation with speed/accel control
  • Smart Typing auto-space & capitalization
  • macOS mode with correct HID mapping
  • Slide-typing, multi-touch, modifier keys
